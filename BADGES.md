## 📝 Instructions d'utilisation

1. Remplacez `VOTRE_USERNAME` par votre nom d'utilisateur GitHub
2. Remplacez `boltai` par le nom de votre repository
3. Copiez les badges dans votre README.md
4. Ajustez les couleurs et styles selon vos préférences

## 🎯 Conseils

- Utilisez des badges cohérents avec votre design
- Ne surchargez pas votre README avec trop de badges
- Privilégiez les badges les plus importants
- Testez les liens de déploiement avant de publier
