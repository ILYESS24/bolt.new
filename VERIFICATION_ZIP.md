## 🚀 **Instructions d'Utilisation**

### **1. Téléchargement**
```bash
# Le fichier ZIP est prêt à télécharger
boltai-complete-project.zip (945KB)
```

### **2. Extraction**
```bash
unzip boltai-complete-project.zip
cd boltai-complete-project
```

### **3. Installation**
```bash
npm install
```

### **4. Configuration**
```bash
cp .env.example .env.local
# Éditer .env.local avec vos clés API
```

### **5. Démarrage**
```bash
npm run dev
```

### **6. Déploiement**
```bash
# Vercel (recommandé)
npm run deploy:vercel

# Ou autre plateforme
npm run deploy:railway
npm run deploy:render
```
